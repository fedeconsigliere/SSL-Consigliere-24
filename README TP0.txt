El entorno de desarrollo elegido es Visual Studio Code
El compilador seleccionado para los programas en C, es GCC, encontramos la version instalada tirando en el CMD de windows gcc --version. En este caso es la GCC 11.2.0
Vamos estar usando para compilar la versión C18 de windows, brindandole el comando -std
Codeamos el código de hola mundo dentro de nuesto IDE
Posterior con los comandos de gcc compilamo nuestro programa "gcc HolaMundo.c -o a.out" (Cabe aclarar que si no usamos "-o a.out" gcc automaticamente lo denominaba asi, por ende no haria falta, excepto que se le quiera poner otro nombre)
Probamos la ejecucion del programa con el comando ".\a.exe" en nuestra terminal
Ejecutamos en la terminal el comando "./a.exe > output.txt" para que lo que nuestro programa genera sea impreso en un archivo txt denominado output
Luego iniciamos una terminal bash en nuestro mail
Inicializamos el git "git init"
despues seleccionamos todos nuestros archivos "git add ."
Luego los commiteamos con "git commit"
